Je suis ✨ inarrêtable ✨ (psst, sisi il s'arrête mais c'est un secret)

Inarrêtable, c'est une boucle qui va te répeter qui il est, mais en pleins de languages différents. Exemple :

// Inarrêtable, Swift (aka le premier inarrêtable)
while true {
  print("JE SUIS INARRÊTABLE")
}
// Inarrêtable, JavaScript
while (true){
  console.log("JE SUIS INARRÊTABLE")
}
# Inarrêtable, Python
while True:
  print("JE SUIS INARRÊTABLE")
# Inarrêtable, Ruby
while true
  puts("JE SUIS INARRÊTABLE, et aussi dans 12 autres languages")
end
